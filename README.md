# Student Manager (WinForms)

Простое WinForms-приложение для управления списком студентов.

## Возможности
- Добавление студента (отдельная форма)
- Удаление с подтверждением
- Поиск по:
  - имени
  - группе
  - возрасту
- Сохранение и загрузка данных в JSON
- Привязка через BindingList и BindingSource

## Технологии
- C#
- .NET
- WinForms
- LINQ
- System.Text.Json
